## Two lines of code
<Note>We provide a white-glove service to help you deploy. Interested? Just reach out <a href="mailto@support@appliedlabs.ai">here</a></Note>
Add the following two lines of code to your HTML, Next.js, React, or any web framework of your choice to render the chat. 
```tsx
<Script src="https://appliedlabs.ai/cdn/applied-chat.umd.js" />
<applied-chat agent-id={agentId}/>
```

## Setting user details
We aggregate conversations and tag based on the passed in user ID. Addtionally, metadata fields such as `name`, `email`, `phone` are associated with the user.
```tsx
<applied-chat 
  agent-id={agentId} 
  user-id={yourUserId}
  metadata={{
    name: "Alice", 
    email: "alice@wonderland.com",
    coins: 100,
    // any other fields you want
  }}
/>
```
You can also change the metadata using
```tsx
window.applied.setMetadata({
  name: "Hatter",
  email: "hatter@wonderland.com",
  coins: -5,
})
```

## To hide and show the chat
Call the hide and show functions
```tsx
window.applied.hide()
window.applied.show()
```

## Full page embed
Style the parent container to be large and place it anywhere you want. We will render the chat inside the container.
```tsx
<div style={{height: "100%", width: "100%"}}>
  <applied-chat agent-id={agentId} size="full"/>
</div>
```
